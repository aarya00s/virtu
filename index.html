Predicting CS:GO round winners

Optimizing Sports Betting for CS:GO

This project explores the application of machine learning to predict outcomes in CS:GO matches. Utilizing a dataset from professional games from Kaggle, it aims to create a predictive system, leveraging data preprocessing and feature engineering.

Through methodologies such as Logistic Regression and Random Forests, alongside domain knowledge of CS:GO's gameplay and economy, the project analyzes match data to forecast winners. Challenges include the dynamic nature of matches and the importance of momentum.

Features

  • Data Source: Uses a dataset from Kaggle's CS:GO Round Winner Classification.
  • Technologies: Python, Pandas, Matplotlib, Seaborn, Scikit-learn.
  • Models Used: Logistic Regression, Decision Trees, Random Forest.
  • Approach: Combines data preprocessing, feature engineering, and machine learning for prediction accuracy.

Despite the unpredictability of CS:GO, the project demonstrates the potential of machine learning in esports betting strategies, offering insights into match dynamics and enhancing prediction accuracy upto 87%.
